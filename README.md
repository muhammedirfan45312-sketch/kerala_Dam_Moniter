# 🌊 DamGuard Kerala: AI-Driven Reservoir Telemetry & Flood Early Warning System

**Live Deployment:** [https://kerala-dam-moniter.vercel.app](https://kerala-dam-moniter.vercel.app)

DamGuard Kerala is a high-precision, premium dashboard designed for real-time reservoir monitoring and autonomous flood risk mitigation. Built in response to the critical need for advanced early warning systems, it integrates live telemetry, AI-driven decision support, and multi-agency alert simulation.

## 🚀 Key Features

### 🤖 AI Autonomous Control Center
- **Neural Risk Assessment:** Real-time analysis of inflow, capacity, and precipitation clusters.
- **Operator Consultation:** A fail-safe "Human-in-the-loop" authorization protocol for dam gate operations.
- **Live Gate Visualizer:** Animated spillway shutter control simulation with feedback on discharge rates.

### 🛡️ Early Warning & Crisis Management
- **Alert Notification Simulator:** Broadcast emergency alerts to 6 key agencies (Kerala Police, Fire & Rescue, Village Officers, etc.) with a single click.
- **Downstream Vulnerability Tracking:** Per-dam downstream impact zone analysis and Malayalam-language warning distribution.
- **IMD Integration:** Real-time weather alert overlays (Orange/Red/Yellow) based on catchment area forecasts.

### 🗺️ Interactive Kerala Dam Network
- **Dynamic Geo-Mapping:** Leaflet-powered map of Kerala featuring all 18 major KSEB and Irrigation dams.
- **Telemetry Dashboards:** Dedicated intelligence panels for every dam, providing storage %, inflow (cumecs), and outflow dynamics.

### 📰 Live Intelligence Bulletin
- **Breaking News Ticker:** Scrolling real-time headlines verified as of March 2026.
- **Situational Reports:** Integrated bulletins from KSDMA, CWC, and PRD Kerala.

## 🛠️ Tech Stack
- **Frontend:** Vanilla HTML5, CSS3 (Custom Design System), JavaScript (ES6+)
- **Mapping:** Leaflet.js (Dark Mode Tiles)
- **Deployment:** Vercel
- **Version Control:** Git
